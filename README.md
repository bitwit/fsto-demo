## Install

```sh
$ npm install --save fsto-demo
```


## Usage

```js
var fsto-demo = require('fsto-demo');

fsto-demo('Rainbow');
```

```sh
$ npm install --global fsto-demo
$ fsto-demo --help
```

```sh
# creates a browser.js
$ npm run browser
```
